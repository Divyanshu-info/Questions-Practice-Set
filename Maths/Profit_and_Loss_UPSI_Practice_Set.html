  • (B) ₹580
  • (C) ₹600

विस्तृत हल

1. हल (B)

माना क्रय मूल्य (CP) = x

विक्रय मूल्य (SP) = CP + लाभ = x + 0.125x = 1.125x

1.125x = 630

x = 630 / 1.125 = ₹560

2. हल (C)

माना 1 सेब का क्रय मूल्य CP है और 1 सेब का विक्रय मूल्य SP है।

15 × CP = 25 × SP

CP / SP = 25 / 15 = 5 / 3

हानि = CP - SP = 5 - 3 = 2

हानि % = (हानि / CP) × 100 = (2 / 5) × 100 = 40%

3. हल (B)

माना क्रय मूल्य = ₹100

अंकित मूल्य = 100 + 30% of 100 = ₹130

छूट = 10% of 130 = ₹13

विक्रय मूल्य = 130 - 13 = ₹117

लाभ = 117 - 100 = ₹17

लाभ % = 17%

4. हल (B)

कुल क्रय मूल्य = 1500 + 1500 = ₹3000

पहली घड़ी पर लाभ = 12% of 1500 = ₹180

दूसरी घड़ी पर लाभ = 18% of 1500 = ₹270

कुल लाभ = 180 + 270 = ₹450

कुल लाभ % = (450 / 3000) × 100 = 15%

5. हल (C)

माना क्रय मूल्य = x

10% हानि पर विक्रय मूल्य = 0.90x = 72 => x = 72 / 0.9 = ₹80

5% लाभ के लिए नया विक्रय मूल्य = 80 + 5% of 80 = 80 + 4 = ₹84

6. हल (B)

त्रुटि = 1000 ग्राम - 900 ग्राम = 100 ग्राम

लाभ % = (त्रुटि / सही मान - त्रुटि) × 100

लाभ % = (100 / 900) × 100 = 11.11%

7. हल (C)

पहली छूट के बाद मूल्य = 800 - 10% of 800 = 800 - 80 = ₹720

दूसरी छूट = 720 - 612 = ₹108

दूसरी छूट % = (108 / 720) × 100 = 15%

8. हल (B)

माना A का क्रय मूल्य = x

B का क्रय मूल्य = 1.10x

C का क्रय मूल्य = 1.20 × (1.10x) = 1.32x

1.32x = 264 => x = 264 / 1.32 = ₹200

9. हल (B)

100 संतरों का क्रय मूल्य = ₹350 => 1 संतरे का CP = ₹3.5

1 दर्जन (12) संतरों का विक्रय मूल्य = ₹48 => 1 संतरे का SP = ₹4

लाभ = 4 - 3.5 = ₹0.5

लाभ % = (0.5 / 3.5) × 100 = 14.28%

10. हल (C)

माना क्रय मूल्य (CP) = x, तो विक्रय मूल्य (SP) = 1.2x

नया CP = x - 100, नया SP = 1.2x - 100

नया लाभ % = 20% + 4% = 24%

(नया SP - नया CP) / नया CP = 0.24

(1.2x - 100 - (x - 100)) / (x - 100) = 0.24

0.2x / (x - 100) = 0.24 => 0.2x = 0.24x - 24

0.04x = 24 => x = ₹600

11. हल (B)

जब दो वस्तुओं का विक्रय मूल्य समान हो और एक पर x% लाभ और दूसरे पर x% हानि हो, तो हमेशा हानि होती है।

हानि % = (x/10)² = (25/10)² = 6.25%

कुल विक्रय मूल्य = 1875 + 1875 = ₹3750

कुल क्रय मूल्य (CP) = 3750 / (1 - 0.0625) = 3750 / 0.9375 = ₹4000

कुल हानि = 4000 - 3750 = ₹250

12. हल (B)

क्रय मूल्य (CP) = ₹1200, लाभ = 10%

विक्रय मूल्य (SP) = 1200 × 1.10 = ₹1320

माना अंकित मूल्य (MP) = x

छूट के बाद SP = MP × (1 - छूट%) = x × 0.80

0.80x = 1320 => x = 1320 / 0.80 = ₹1650

13. हल (C)

5 केलों का क्रय मूल्य = ₹4 => 1 केले का CP = 4/5 = ₹0.80

4 केलों का विक्रय मूल्य = ₹5 => 1 केले का SP = 5/4 = ₹1.25

लाभ % = (0.45 / 0.80) × 100 = 56.25%

14. हल (D)

माना विक्रय मूल्य (SP) = ₹100, तो लाभ = ₹25

क्रय मूल्य (CP) = SP - लाभ = 100 - 25 = ₹75

वास्तविक लाभ % = (लाभ / CP) × 100 = (25 / 75) × 100 = 33.33%

15. हल (B)

16. हल (C)

छूट के बाद SP = ₹90

CP पर 8% लाभ है, तो SP = CP × 1.08

90 = CP × 1.08 => CP = 90 / 1.08 = ₹83.33

यदि कोई छूट नहीं है, तो SP = MP = ₹100

नया लाभ = 100 - 83.33 = ₹16.67

नया लाभ % = (16.67 / 83.33) × 100 = 20%

17. हल (B)

18. हल (B)

25. हल (B)

जब दो वस्तुओं का विक्रय मूल्य समान हो और एक पर x% लाभ और दूसरे पर x% हानि हो, तो हमेशा (x²/100)% की हानि होती है।

27. हल (B)

कुल CP = (10 × 10) + (15 × 12) = 100 + 180 = ₹280

कुल मात्रा = 10 + 15 = 25 किलो

कुल SP = 25 × 14 = ₹350

माफ कीजिए, गणना में त्रुटि हुई। सही गणना है:

लाभ % = (70 / 280) * 100 = 25% नहीं, बल्कि (70/280) * 100 = 1/4 * 100 = 25% होता है।

चलिए फिर से गणना करते हैं।
