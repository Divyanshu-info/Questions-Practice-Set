भाग B: क्रमिक हल (Step-by-Step)
नीचे प्रत्येक प्रश्न का संक्षिप्त, सुसंगत, जाँच योग्य समाधान दिया गया है।
1) उत्तर: ₹24,000 (D)
मानें भाग P1, P2; P1+P2=48000. SI समता: P1·12·4/100 = P2·16·3/100 ⇒ 48P1 = 48P2 ⇒ P1=P2=24000. अतः दूसरे भाग = ₹24,000।
2) उत्तर: 5% (B)
SI = (1/4)P = P·R·T/100 और R=T ⇒ R²/100 = 1/4 ⇒ R²=25 ⇒ R=5%।
3) उत्तर: ₹12,000 (B)
दोनों बराबर राशियाँ = A. कुल SI = A·9·5/100 + A·13·5/100 = 1.1A. 1.1A = 13200 ⇒ A = 12000।
4) उत्तर: ₹28,800 (C)
कुल प्रतिशत = 3×8 + 4×10 + 3×12 = 24+40+36 = 100% ⇒ 10 वर्षों में SI = P. दिया SI = 28800 ⇒ P = 28800।
5) उत्तर: 12% (C)
SI = 36000·r·6/100 = 2160r = 25920 ⇒ r = 12%।
6) उत्तर: ₹30,000 (C)
समान SI ⇒ P ∝ 1/दर ⇒ 1/5 : 1/6 : 1/10 = 6:5:3 (LCM तरीका). योग 14 भाग ⇒ कुल 84000 ⇒ 1 भाग = 6000 ⇒ 6% वाला (5 भाग) = 30000।
7) उत्तर: 24 वर्ष (B)
12 वर्ष में SI = P ⇒ 1 वर्ष में SI = P/12. तीन गुना हेतु SI = 2P ⇒ समय = (2P)/(P/12) = 24 वर्ष।
8) उत्तर: ₹1,00,000 (C)
SI = A·8·2/100 + A·12·3/100 = 0.52A = 26000 ⇒ A = 50000. कुल निवेश = 2A = ₹1,00,000।
9) उत्तर: ₹35,000 (B)
4y,10% पर SI=14000 ⇒ P = 14000·100/(10·4)=35000. अगले 4y 15% ⇒ SI = 35000·15·4/100=21000. कुल = 14000+21000=₹35000।
10) उत्तर: ₹1,20,000 (B)
आय = P/3·7% + P/4·9% + (P−P/3−P/4)·12%
= 0.023333P + 0.0225P + (5/12)·0.12P = (0.023333+0.0225+0.05)P = 0.095833P.
0.095833P = 11500 ⇒ P = 120000।
11) उत्तर: 10% (D)
2 वर्षों का अतिरिक्त = 72000−62400 = 9600 ⇒ 1 वर्ष का SI = 4800.
P = 62400 − 3×4800 = 48000 ⇒ R = 4800·100/48000 = 10%।
12) उत्तर: ₹25,000 (C)
कुल SI = A·8·4/100 + A·11·4/100 = 0.76A = 19000 ⇒ A = 25000 (हर ऋण)।
13) उत्तर: 15% (C)
SI = P·R·T/100 ⇒ 12000 = 20000·R·4/100 ⇒ 12000 = 800R ⇒ R = 15%।
14) उत्तर: 10% (C)
CI−SI (2y) = P·(R/100)² = 500 ⇒ 50000·R²/10000 = 500 ⇒ R²=1 ⇒ R=10%।
15) उत्तर: ₹1,04,000 (B)
समय: छोटे के लिए 7y, बड़े के लिए 3y. समान परिपक्वता: 1.7P1 = 1.3P2 ⇒ P1/P2 = 13/17. कुल = 240000 ⇒ P1 = (13/30)·240000 = 104000।
16) उत्तर: ₹27,000 (B)
समान वार्षिक SI ⇒ P ∝ 1/दर ⇒ 1/3 : 1/5 : 1/9 = 15:9:5 (योग 29). 5% भाग = 9/29 × 87000 = 27000।
17) उत्तर: 45 वर्ष (B)

मानें x 14% पर, (120000−x) 10% पर, 2y ⇒ 0.2(120000−x) + 0.28x = 28800 ⇒ 24000 − 0.2x + 0.28x = 28800 ⇒ 0.08x = 4800 ⇒ x = 60000।

26) उत्तर: 4% (B)
नया लक्ष्य: 5000 = 25000·R·5/100 ⇒ 5000 = 125R ⇒ R = 4%।
27) उत्तर: ₹10,500 (B)
समता: P1(1+0.1·2) = P2(1+0.1·4) = P3(1+0.1·5) ⇒ 1.2P1 = 1.4P2 = 1.5P3.
P ∝ 1/(1+rt) ⇒ 1/1.2 : 1/1.4 : 1/1.5 = (5/6):(5/7):(2/3) ⇒ 35:30:28 (LCM 210). योग 93 भाग ⇒ कुल 27900 ⇒ 1 भाग = 300 ⇒ P1 = 35×300 = 10500।

30) उत्तर: 10% (A)
SI 3y = 7200 ⇒ 1y = 2400 ⇒ 2y SI = 4800. दिया 2y CI = 5040 ⇒ अंतर = 240 = P·(R/100)².
